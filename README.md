# MERN Portfolio Website

A full-stack portfolio website built with the MERN stack (MongoDB, Express, React, Node.js).

## Features

- Personal profile and bio
- Projects showcase with GitHub and live demo links
- Contact form that stores messages in MongoDB
- Responsive design with Tailwind CSS
- RESTful API for data management
- MongoDB Atlas integration

## Tech Stack

- **Frontend**: React (Vite + JSX)
- **Styling**: Tailwind CSS (via CDN)
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas
- **HTTP Client**: Axios
